Portal for Medical physics Quality Assurance

This web application was designed with the aim of making the job of the medical physicist easy. All the software tools used by the medical physicist will has been centralized in a single application. The application is designed with scalability in mind and incorporates python packages such as the Pylinac package and packages developed in-house. The application is designed to serve as a repository and a data center for all quality control tests performed by medical physics at the hospital. This application also contains the medical physics standard operating procedures and these have been made to be available on demand.

The implementation of the application will enable the institution to continue in the direction of a paperless department and will also improve the ability of the medical physics department to track the performance of the machines.
